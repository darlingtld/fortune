1. install mysql
    use db_init.sql to initialize the database
    configuration file is located at fortune.properties

2. localhost:8080/userapp/druid
    open database monitoring stuff provided by druid

3. install memcached (mac)
    use 'brew install memcached'
    start it up with '/usr/local/opt/memcached/bin/memcached'
    default port is 11211
    configuration file is located at fortune.properties

    for windows, refer to https://commaster.net/content/installing-memcached-windows
    start memcache via running 'memcached.exe'

4. install mongodb
    refer to http://www.mongoing.com/downloads

    create a database named 'fortune'

    mongodump -h dbhost -d dbname -o dbdirectory

    mongorestore -host localhost:27017 -d fortune /dir dbdirectory
    
    (just for frank):mongod --dbpath /Users/frank/software/mongodb/dbs/

5. configure kafka
    refer to http://kafka.apache.org/documentation.html#quickstart

    bin/zookeeper-server-start.sh config/zookeeper.properties

    cp config/server.properties config/server-1.properties
    cp config/server.properties config/server-2.properties
    config/server-1.properties:
        broker.id=1
        port=9093
        log.dir=/tmp/kafka-logs-1

    config/server-2.properties:
        broker.id=2
        port=9094
        log.dir=/tmp/kafka-logs-2

    bin/kafka-server-start.sh config/server-1.properties &

    bin/kafka-server-start.sh config/server-2.properties &

    bin/kafka-topics.sh --create --zookeeper localhost:2181 --replication-factor 3 --partitions 5 --topic lingda-test

    bin/kafka-topics.sh --describe --zookeeper localhost:2181 --topic lingda-test

6. install redis
    refer to http://redis.io/
    for windows, https://github.com/MSOpenTech/redis/releases
    run with 'redis-server'



Mark Six rules
四、  以下所有投注皆含本金。
          每期六合彩开奖球数共七粒（最後一码为特码）。

五、  两面:指单、双；大、小；特尾大、特尾小；合单、合双；合尾大、合尾小；家禽、野兽。

5.1    特码单、双∶特码为双数叫特双，如8、16；特码为单数叫特单，如21、35；49则均为和。

5.2    特码大、小∶开出之特码大於或等於25为特码大，小於或等於24为特码小；49则均为和。

5.3    特尾大、小∶开出之特码取出尾数大於或等於5为特尾大，小於或等於4为特尾小；49则均为和。

5.4    特码合单双∶以特码个位和十位数字之和来判断胜负，如01、12、29、32、47为合单；02、11、33、48为合双；49则均为和。

5.5    特码合大小∶以特码个位和十位数字之和来判断胜负，介於1-6为合小；介於7-12为合大；13则为和。
                                 如01、24、33为合小；07、34、48为合大；49则为和。

5.6    合尾大、小∶以特码个位和十位数字之和取出尾数来判断胜负，尾数大於或等於5为合尾大，如14、24、42；
                                 尾数小於或等於4为合尾小，如19、37、49；若特码开出25则为和局。

5.7    家禽、野兽∶开出的特码属於十二生肖中的牛、马、羊、鸡、狗、猪号码为家禽，开出的特码属於十二生肖中的鼠、虎、龙、蛇、兔、猴号码为野兽；
                                 开出49则视为和局。
六、  特码∶香港六合彩公司当期开出的最後一码为特码，假如投注号码为开出之特码，视为中奖，其馀情形视为不中奖。

七、  特大中小∶开出之特码落在1~16为小；17~32为中；33~48为大；49则为和。

八、  正码∶香港六合彩公司当期开出之前6个号码叫正码。每一个号码为一投注组合，假如投注号码为开奖号码之正码，视为中奖，其馀情形视为不中奖。

九、  正码1～6∶香港六合彩公司当期开出之前6个号码叫正码。第一时间出来的叫正码一，依次为正码二、正码三┅ 正码六 （并不以号码大小排序）。
                              正码一、正码二┅ 正码六的大小、单双、色波与合码和特码的大小、单双、色波与合码的规则相同，如正码一为29，则正码一为大、为单；
                              正码二为10，则正码二为小、为双；号码若开出为49，则大小、单双与合码皆视为和局。假如投注组合符合中奖结果，视为中奖，其馀情形视为不中奖。

十、  正码特∶正特一、正特二、正特三、正特四、正特五、正特六∶指下注的正码特与开出之正码其开奖顺序及开奖号码相同，视为中奖，
                          如正码一开出号码8，下注正特一为8者视为中奖，其馀情形视为不中奖。

十一、  总和单双∶所有七个开奖号码的分数总和是单数叫（总分单），如分数总和是115、183；分数总和是双数叫(总分双)，如分数总和是108、162。
                                   假如投注组合符合中奖结果，视为中奖，其馀情形视为不中奖。

十二、  总和大小∶所有七个开奖号码的分数总和大於或等於175为总分大；分数总和小於或等於174为总分小。
                                   如开奖号码为02、08、17、28、39、46、25，分数总和是165，则总分小。
                                  假如投注组合符合 中奖结果，视为中奖，其馀情形视为不中奖。

十三、  连码∶

13.1   三中二∶所投注的每三个号码为一组合，若其中2个是开奖号码中的正码，即为三中二，视为中奖；若3个都是开奖号码中的正码，即为三中二之中三，
                         其馀情形视为不中奖。如06、07、08为一组合，开奖号码中有06、07两个正码，没有08，即为三中二，按三中二赔付；
                         如开奖 号码中有06、07、08三个正码，即为三中二之中三，按中三赔付；如出现1个或没有，视为不中奖。

13.2   三全中∶所投注的每三个号码为一组合，若三个号码都是开奖号码之正码，视为中奖，其馀情形视为不中奖。
                         如06、07、08三个都是开奖号码之正码，视为中奖，如两个正码加上一个特码视为不中奖。

13.3   二全中∶所投注的每二个号码为一组合，二个号码都是开奖号码之正码，视为中奖，其馀情形（含一个正码加一个特码）视为不中奖。

13.4   二中特∶所投注的每二个号码为一组合，二个号码都是开奖号码之正码，叫二中特之中二；若其中一个是正码，一个是特码，叫二中特之中特；
                          其馀情形视为不中奖。

13.5   特串∶所投注的每二个号码为一组合，其中一个是正码，一个是特码，视为中奖，其馀情形（含二个号码都是正码）视为不中奖。

13.6   自选不中∶所投注的每 5 或 6 或 7 或 8 或 9 或 10 或 11 或 12 个号码为一组合，若当期开奖的六个正码和特码的号码都不在该下注组合的号码内，
                              即视为中奖；其馀情形视为不中奖。以五个号码为一组为例，如06、17、28、32、45为一组合，开奖的正码号码为02、05、11、20、25、31，
                              特码为36，没有号码落在该组合内则该组合为中奖；如开奖的正码号码为02、05、11、20、28、31，特码为06，在该组合内出现一个或一个
                              以上的号码，则一律视为不中奖。

十四、  过关∶可同时挑选多种玩法，串联成投注组合，但每注当中同一个号码最多只能勾选两种玩法，只要当期正码所开出之中奖结果完全符合您所选定之串联玩法，
                         即视为中奖。若该期任一组正码中有开出49则过关即视为和局。
                         ※注∶正码过关每注最高派彩为 RMB$500000

十六、  半波∶以特码色波和特单、特双、特大和特小为一个投注组合，当期特码开出符合投注组合，即视为中奖；若当期特码开出49号则视为和局；其馀情形视为不中奖。

十七、  合肖∶依合肖类别挑选1到11个生肖（排列如同生肖）为一个组合，并选择开奖号码的特码是否在此组合内，若选择【中】且开奖号码的特码亦在此组合内，则视为中奖
                          。若选择【不中】且开奖号码的特码一不在此组合内，即视为中奖，其馀情形视为不中奖；开出49则视为和局。

十八、  一肖(A/B)∶指开奖正码及特码７个号码当中含有所下注生肖内的一个或多个号码则视为中奖，但派彩只派一次，即不论同生肖号码出现一个或多个号码都只派彩一次。
                                   今年为羊年，则由羊年为起始生肖，其生肖号码代表请参考「特码 生肖」项目规则。

十九、  尾数(A/B)∶指开奖正码及特码７个号码中含有所下注尾数的一个或多个号码则视为中奖，但派彩只派一次，即不论同生肖号码出现一个或多个号码都只派彩一次。
                                  1尾∶01, 11, 21, 31, 41
                                  2尾∶02, 12, 22, 32, 42
                                  3尾∶03, 13, 23, 33, 43
                                  4尾∶04, 14, 24, 34, 44
                                  5尾∶05, 15, 25, 35, 45
                                  6尾∶06, 16, 26, 36, 46
                                  7尾∶07, 17, 27, 37, 47
                                  8尾∶08, 18, 28, 38, 48
                                  9尾∶09, 19, 29, 39, 49
                                  0尾∶10, 20, 30, 40
                                  ※例如∶开奖结果正码是 11、31、 42、 44、 35、 32 特码21，则1尾2尾4尾5尾都为中奖，其他尾数都不中奖。
                                  　

二十、  连肖∶依选择的连肖类别挑选 2 ～ 5 个生肖（排列如同生肖）为一个组合，当期的开奖号码（六个正码及特码）座落於所勾选之生肖组合所属号码内，
                          且所勾选之生肖皆至少有中一个号码，则视为中奖；其馀情形一律视为不中奖；若选择的生肖组合内有包含本年生肖则称之为正肖，
                          没有包含本年生肖则称之为平肖，正肖与平肖的中奖赔率不同（请注意∶49 亦算输赢，不为和）。

二十一、  总肖∶只要当期号码（所有正码及特码共７个号码）开出的不同生肖总数，与所投注之预计开出之生肖总数符合（不用指定生肖），即视为中奖
                             ，其馀情形视为不中奖。例如当期开奖号码为 19、24、12、34、40、39，特码为49总计开出六个生肖，若下注【总肖6】则为中奖
                             （请注意∶49 亦算输赢，不为和）。

二十二、  总肖单双∶只要当期号码（所有正码及特码共７个号码）开出的不同生肖总数若为单数则为单，若为双数即为双。
                                      例如当期开奖号码为 19、24、12、34、40、39，特码为49总计开出六个生肖，若下注【双】则为中奖。

二十三、  连尾∶挑选2～4个尾数为一组合，当期号码（所有正码与最後出的特码）座落於投注时所勾选之尾数组合所属号码内，则视为中奖，其馀情形视为不中奖
                             (请注意49亦算输赢，不为和)
                             ※举例1∶下注2尾碰，勾选0,1，当期七个号码若0尾及1尾皆有开出，视为中奖。
                             ※举例2∶下注2尾碰，勾选0,1,2三种尾数，会出现三种组合（组合一∶0,1）（组合二∶0,2）（组合三∶1,2），当期七个号码若开出其中一种组合所选的两个尾数
                                               ，视为中奖。
